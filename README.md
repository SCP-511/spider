# spider
爬虫项目
1.主要目标：爬取链家北京地区租房信息
A：分析北京有哪几个区域以及区域下的子区域
B：分析房屋具体信息
C：分析每个区域及其子区域的URL，每个房屋模块信息的具体代码和分页url
D：编写爬虫（scrapy框架，使用随机User-Agent或代理IP池)
E：将数据存入MongoDB中
F：数据处理及可视化
